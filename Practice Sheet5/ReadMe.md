# CodeWithHarry Python Course  
## Practice Sheet 5: Dictionaries and Sets  

This repository contains practice questions from **Practice Sheet 5** of the **CodeWithHarry Python Course**, focusing on **Dictionaries and Sets** in Python.  

---

### Concepts Covered:  
- **Dictionaries**: Learn how to store key-value pairs efficiently.  
- **Dictionary Operations**: Adding, updating, and deleting key-value pairs.  
- **Dictionary Methods**: Explore built-in methods like `keys()`, `values()`, `items()`, `get()`, `update()`, and `pop()`.  
- **Sets**: Understand unordered collections of unique elements.  
- **Set Operations**: Union, intersection, difference, and symmetric difference.  
- **Set Methods**: Using `add()`, `remove()`, `discard()`, `clear()`, and `pop()`.  
- **Dictionary vs Set**: Key differences and when to use each.
