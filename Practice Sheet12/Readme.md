# CodeWithHarry Python Course
## Practice Sheet 12: Advanced Python - Part 1

This repository contains solutions and practice questions from Practice Sheet 12 of the CodeWithHarry Python Course, focusing on advanced Python concepts.  

---

### **Concepts Covered**  
- **Walrus Operator**: Assign values within expressions (`:=`).  
- **Type Hints**: Add type annotations for variables and functions.  
- **Advanced Type Hints**: Use `List`, `Tuple`, `Dict`, `Union` from the `typing` module.  
- **Match-Case**: Pattern matching introduced in Python 3.10.  
- **Dictionary Merge & Update Operators**: Use `|` and `|=` for merging/updating dictionaries.  
- **Exception Handling**: Use `try`, `except`, `else`, and `finally` for error handling.  
- **Raise Exceptions**: Create custom exceptions with `raise`.  
- **`if __name__ == "__main__"`**: Check if script is run directly or imported.  
- **Global Keyword**: Modify variables outside the current scope.  
- **Enumerate Function**: Iterate with index using `enumerate()`.  
- **List Comprehensions**: Create lists in a concise way.
