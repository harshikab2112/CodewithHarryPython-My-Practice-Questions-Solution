# CodeWithHarry Python Course
## Practice Sheet 12: Advanced Python - Part 1

This repository contains solutions and practice questions from Practice Sheet 12 of the CodeWithHarry Python Course, focusing on advanced Python concepts.  

---

### **Concepts Covered**  
- **List Comprehensions**: Creating concise and efficient lists.  
- **Generators and Iterators**: Understanding lazy evaluation and iterable objects.  
- **Decorators**: Enhancing function behavior using `@decorator` syntax.  
- **Context Managers**: Efficient resource management using `with` statements.  
- **Regular Expressions**: Pattern matching with the `re` module.  
- **Metaprogramming**: Dynamic code modification using metaclasses.
