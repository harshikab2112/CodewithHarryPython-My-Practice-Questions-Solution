# CodeWithHarry Python Course
## Practice Sheet 9: File I/O

This repository contains solutions and practice questions from Practice Sheet 9 of the CodeWithHarry Python Course, focusing on File Input and Output (I/O) in Python.  

---

### **Concepts Covered**  
- **Reading Files**: Using `open()`, `read()`, `readline()`, and `readlines()`.  
- **Writing to Files**: Writing data using `write()` and `writelines()`.  
- **File Modes**: Understanding read (`r`), write (`w`), append (`a`), and exclusive creation (`x`) modes.  
- **Working with File Paths**: Handling absolute and relative paths.  
- **Using `with` Statement**: Ensuring proper file handling with context managers.
