# CodeWithHarry Python Course
## Practice Sheet 6: Conditional Statements

This repository contains solutions and practice questions from **Practice Sheet 6** of the CodeWithHarry Python Course, focusing on **Conditional Statements** in Python.

---

## Concepts Covered
- Understanding **if, elif, else** statements for decision-making.
- **Nested Conditions**: Handling multiple conditions within conditions.
- **Logical Operators**: Utilizing `and`, `or`, and `not` for complex conditions.
- **The match Statement**: Using the `match` statement (Python 3.10+) for pattern matching.
