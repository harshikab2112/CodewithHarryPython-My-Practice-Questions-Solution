# CodeWithHarry Python Course
## Practice Sheet 10: Object-Oriented Programming (OOP)

This repository contains solutions and practice questions from Practice Sheet 10 of the CodeWithHarry Python Course, focusing on Object-Oriented Programming (OOP) in Python.  

---

### **Concepts Covered**  
- **Classes and Objects**: Understanding how to create and use classes and objects.  
- **Constructors (`__init__` method)**: Initializing object attributes.  
- **Instance and Class Variables**: Differentiating between instance and class-level data.  
- **Methods**: Creating instance methods, class methods, and static methods.  
- **Encapsulation**: Using access specifiers (`public`, `_protected`, `__private`).  
- **Inheritance**: Implementing single and multiple inheritance.  
- **Polymorphism**: Understanding method overloading and overriding.  
- **Abstraction**: Using abstract base classes to define interfaces.
