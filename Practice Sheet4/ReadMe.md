## Practice Sheet 4: Lists and Tuples  

This repository contains practice questions from **Practice Sheet 4** of the **CodeWithHarry Python Course**, focusing on **Lists and Tuples** in Python.  

---

### Concepts Covered:  
- **Lists**: Learn how to store and manipulate collections of items.  
- **List Operations**: Indexing, slicing, concatenation, and repetition.  
- **List Methods**: Explore built-in methods like `append()`, `extend()`, `insert()`, `remove()`, `pop()`, `sort()`, and more.  
- **Tuples**: Understand immutable sequences and how they differ from lists.  
- **Tuple Operations**: Indexing, slicing, and tuple unpacking.  
- **Tuple Methods**: Using `count()` and `index()`.  
- **List vs Tuple**: Key differences and when to use each.
