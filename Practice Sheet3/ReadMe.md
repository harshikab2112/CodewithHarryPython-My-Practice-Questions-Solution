# CodeWithHarry Python Course
## Practice Sheet 3: Strings

This repository contains practice questions from **Practice Sheet 3** of the CodeWithHarry Python Course, focusing on **Strings** in Python.

### Concepts Covered:
- **Strings:** Learn how to work with text data in Python.
- **String Operations:** Concatenation, repetition, slicing, and indexing.
- **String Methods:** Explore built-in methods like `lower()`, `upper()`, `replace()`, `strip()`, and more.
- **String Formatting:** String interpolation with f-strings, `format()`, and `%` formatting.
- **Escaping Characters:** Working with special characters using escape sequences like `\n`, `\t`, and `\\`.
